== Description ==
Automatically sync posts and featured images between two WordPress sites in real-time using the REST API. Developed by Ammar Rashad.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/sync-posts-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure your settings through the 'Sync Posts' menu.

== Frequently Asked Questions ==
= Is it safe? =
Yes, all communication is protected using WordPress application passwords.

= Does it sync pages too? =
Currently, it is focused on posts only.
